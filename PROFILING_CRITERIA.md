## 1. 프로파일 차원 정의

### 1.1 정치 성향 (Political Stance)

**스케일: 1-10** (1 = 진보, 5 = 중립, 10 = 보수)

| 구간 | 성향 레이블 | 뉴스 프레이밍 방식 |
|------|------------|------------------|
| 1-3 | 진보 | 분배·복지·노동권 중심 프레이밍, 대기업·재벌 비판적 시각 포함 |
| 4-6 | 중도 | 균형 잡힌 복수 시각 제공, 이해관계자 다각도 분석 |
| 7-10 | 보수 | 성장·규제완화·시장 효율 중심 프레이밍, 기업 경쟁력 강조 |

**적용 범주:** 경제, 정치, 사회, 부동산, 기업 관련 기사

**세부 적용 방식:**
- **금리 인상 기사 (진보 1-3):** "서민 대출 이자 부담 증가, 취약계층 영향 집중 분석"
- **금리 인상 기사 (중도 4-6):** "물가 안정 vs 경기 둔화, 양면 효과 균형 분석"
- **금리 인상 기사 (보수 7-10):** "인플레이션 억제와 통화 신뢰성 회복, 기업 투자환경 영향"

> 주의: 정치 성향은 사실 왜곡이 아닌 '어떤 측면을 먼저, 더 깊이 다루는가'의 프레이밍 조정임

### 1.3 문체 선호 (Writing Style)

| 스타일 코드 | 명칭 | 특징 | 문장 예시 |
|-----------|------|------|---------|
| `formal` | 격식체 | 존댓말, 완전한 문장, 수동태 활용 | "한국은행이 기준금리를 0.25%p 인상하였습니다. 이에 따른 시장 반응을 분석해 드리겠습니다." |
| `casual` | 비격식체 | 친근한 어투, 구어체 혼용 | "한은이 금리를 또 올렸어요! 이게 내 통장에 어떤 영향을 줄지 같이 알아봐요." |
| `concise` | 간결체 | 핵심만 압축, 불릿 포인트 활용 | "• 한은 기준금리 +0.25%p\n• 대출이자 월 약 2만원 증가 (1억 기준)\n• 예금금리도 동반 상승 예상" |
| `detailed` | 상세체 | 배경 설명 포함, 맥락 풍부, 용어 해설 | "한국은행 금융통화위원회가 이번 회의에서... (배경 설명) ... 이는 경제학에서 말하는 '테일러 준칙'에 따른 결정으로..." |

---

### 1.4 지식 수준 (Knowledge Level)

| 레벨 | 레이블 | 설명 깊이 | 전문 용어 처리 | 배경 설명 |
|------|-------|---------|-------------|---------|
| `expert` | 전문가 | 결론 및 함의 중심 | 용어 그대로 사용 | 생략 (알고 있다고 가정) |
| `general` | 일반인 | 핵심 + 간단한 맥락 | 용어 사용 후 한 줄 설명 | 1-2문장 요약 |
| `beginner` | 입문자 | 기초부터 단계별 설명 | 쉬운 말로 풀어쓰기 | 충분한 배경 제공, 비유 활용 |

**전문가 vs 입문자 비교 예시 (기준금리 기사):**

- **전문가:** "0.25%p 인상으로 실효 연방기금금리와의 차이가 축소됨. 장단기 금리 역전 현상 완화 가능성 및 원화 강세 압력 주목."
- **일반인:** "기준금리가 0.25% 올랐습니다. 주택담보대출 이자가 조금 더 오를 수 있어요."
- **입문자:** "기준금리란 은행들이 돈을 빌릴 때 내는 이자율이에요. 이게 오르면 우리가 은행에서 대출받을 때 이자도 같이 올라가는 거예요. 쉽게 말해 돈 빌리는 비용이 비싸지는 거죠."

### 1.5 연령대 (Age Group)

| 코드 | 연령대 | 주요 관심사 | 활용 레퍼런스 스타일 | 예시 맥락 |
|------|-------|-----------|-----------------|---------|
| `10s` | 10대 | 입시, 진로, 문화, IT | MZ 트렌드, 게임, 유튜버 비유 | "수능 점수처럼 생각하면..." |
| `20s` | 20대 | 취업, 재테크 입문, 문화 | 사회초년생 경험, 첫 월급, 카카오·네이버 예시 | "첫 월급 받았을 때처럼..." |
| `30s` | 30대 | 내 집 마련, 육아, 커리어 | 주택담보대출, 어린이집, 승진 | "아파트 매수 고민 중이라면..." |
| `40s` | 40대 | 자산 관리, 자녀 교육, 중간 관리자 | 학원비, 노후 준비 시작, 임원 직전 | "자녀 대학 등록금 준비하는 입장에서..." |
| `50s` | 50대 | 은퇴 준비, 노후 자산 | 연금, 퇴직금, 건강 | "은퇴 5년 앞두고 있다면..." |
| `60plus` | 60대+ | 건강, 연금, 생활비 | 노인 복지, 의료비, 경로당 | "고정 수입으로 생활하는 분들은..." |

### 1.6 직업군 (Occupation)

| 코드 | 직업군 | 뉴스 하이라이트 포인트 | 관심 액션 |
|------|-------|-------------------|---------|
| `student` | 학생 | 장학금, 취업 트렌드, 인턴십, 기업 문화 | 지원 방법, 준비 방법 |
| `office_worker` | 직장인 | 연봉 협상, 기업 실적, 노동법 변화 | 이직 신호, 업무 영향 |

| `job_seeker` | 구직자 | 채용 트렌드, 기업 건전성, 업종 전망 | 지원 적합성 |

```

**프로파일링 활용:**
- 사회(17%) 기사는 전 직업군·연령대에 기본 노출
- 기업(14.9%) 기사는 직장인·투자자·구직자 우선 노출
- 증권(6.8%) 기사는 투자자에게 가중치 3배 부여
- 부동산(4.7%) 기사는 30-40대 직장인·투자자 우선

### 2.2 기사 태그 패턴

| 태그 | 건수 | 프로파일 활용 방식 |
|------|------|-----------------|
| `[포토]` | 415건 | 시각적 컨텐츠 선호 사용자 가중 노출 |
| `[속보]` | 104건 | 알림 설정한 관심 분야에 즉시 푸시 |
| `[단독]` | 69건 | 고관심도 사용자 우선 노출 (지식수준 general+) |

**속보 처리 전략:**
- 관심 분야에 해당하는 `[속보]` 기사는 지식 수준·문체 필터 없이 즉시 알림
- `[단독]` 기사는 해당 분야 관심도 상위 20% 사용자에게만 알림

### 2.3 기사 길이 분포

| 지표 | 값 | 프로파일 활용 |
|------|---|------------|
| 본문 중앙값 | 903자 | 간결체 선호자 → 이 길이 내 요약 |
| 본문 평균 | 1,059자 | 상세체 선호자 → 원문 + 추가 맥락 |
| 최소 분포 | ~200자 | 단신 기사 → 모든 프로파일 동일 처리 |
| 최대 분포 | ~5,000자+ | 장문 기사 → 지식수준별 차등 요약 |

**길이별 처리 전략:**
- **200자 미만 (단신):** 프로파일 무관 원문 그대로
- **200-900자 (표준):** 문체 변환만 적용
- **900-2,000자 (중간):** 문체 + 지식수준 적용
- **2,000자 초과 (장문):** 모든 프로파일 차원 완전 적용

### 2.4 댓글 보유 기사 (19%)

댓글이 있는 기사 = **높은 사회적 관심도**를 나타내는 신호

**활용 방식:**
- 댓글 보유 기사는 '사회적 반응 요약' 섹션 추가 가능
- 의견 다양성이 높은 기사 → 정치 성향 필터 더 세밀히 적용
- 직업군·연령대별 반응 편차 분석에 활용

### 2.5 이미지 보유율 (92%)

92%의 기사가 이미지를 보유 → 이미지 기반 관심도 신호 활용 가능

**프로파일링 연계:**
- 이미지 heavy 기사 → 시각 정보 선호 사용자 우선 큐레이션
- 인포그래픽 포함 기사 → 입문자(beginner) 수준 사용자에게 적합

### 2.6 인쇄판 vs 웹 비율 (20% vs 80%)

| 구분 | 비율 | 특성 | 타겟 프로파일 |
|------|------|------|------------|
| 인쇄판 | 20% | 심층 분석, 장문, 전문성 | 전문가 지식수준, 격식체, 50-60대 |
| 웹전용 | 80% | 속보성, 짧은 호흡, 멀티미디어 | 일반·입문자, 간결체, 20-30대 |

## 3. 온보딩 질문 설계

온보딩은 5개 질문으로 구성. 설문 느낌이 아닌 **대화형 인터뷰** 방식.

### Q1. 관심 분야 파악 (Interest Categories)

> **"오늘 아침 커피 마시면서 가장 먼저 보고 싶은 뉴스는?"**

```
[ ] 📈 주식·투자 - 내 자산이 어떻게 움직이는지
[ ] 🏢 기업·비즈니스 - 회사들은 어떻게 돌아가나
[ ] 🏠 부동산 - 집값, 전세, 청약
[ ] 🌍 국제·외교 - 세계가 어떻게 돌아가나
[ ] 💻 IT·테크 - 최신 기술과 디지털 트렌드
[ ] 🎬 문화·연예 - 가볍게 즐기는 소식
[ ] ⚖️ 정치·사회 - 우리 사회의 중요한 이슈들

→ 최대 3개 선택 가능
→ 프로파일 매핑: interest_categories[] (1순위/2순위/3순위)
```

---

### Q2. 직업·생활 상황 파악 (Occupation)

> **"뉴스를 읽을 때 가장 먼저 드는 생각은?"**

```
[ ] 💼 "이게 회사 업무에 어떤 영향을 줄까?" → office_worker
[ ] 📚 "취업·진로에 도움이 되는 정보가 있을까?" → student / job_seeker
[ ] 💰 "투자 포트폴리오에 어떤 영향일까?" → investor
[ ] 🏪 "우리 가게 매출에 영향이 있을까?" → self_employed
[ ] 🔬 "내 전문 분야와 어떻게 연결되나?" → professional
[ ] ☕ "은퇴 후 생활에 참고할 게 있을까?" → retired

→ 단일 선택
→ 프로파일 매핑: occupation
```

---

### Q3. 지식 수준 + 문체 파악 (Knowledge Level & Style)

> **"AI가 뉴스를 요약해준다면, 어떤 방식이 가장 편할까요?"**

```
[ ] 🎯 핵심만 딱딱:
    "금리 0.25%p 인상. 대출이자 증가. 예금이자 상승."
    → concise + general

[ ] 📖 친절하게 설명:
    "한은이 금리를 올렸어요. 이게 무슨 말이냐면, 은행에서
     돈 빌리는 비용이 조금 더 비싸진다는 거예요."
    → casual + beginner

[ ] 📰 신문처럼 깔끔하게:
    "한국은행 금융통화위원회는 기준금리를 0.25%p 인상하였으며,
     이에 따른 시장 영향을 분석하면..."
    → formal + general

[ ] 🔍 전문적으로 깊이:
    "금통위 만장일치 인상. 장단기 스프레드 변화 및
     원/달러 환율 영향 분석 필요."
    → concise + expert

→ 단일 선택
→ 프로파일 매핑: writing_style + knowledge_level (조합 선택)
```

### Q4. 연령대 파악 (Age Group)

> **"요즘 가장 신경 쓰이는 삶의 고민은? (솔직하게!)"**

```
[ ] 🎓 대입·진로·첫 사회생활 준비 → 10s / 20s
[ ] 💍 취업·결혼·내 집 마련 고민 → 20s / 30s
[ ] 🏫 자녀 교육비·가족 안정·커리어 도약 → 30s / 40s
[ ] 💪 노후 준비·자산 관리·건강 챙기기 → 40s / 50s

## 4. 개인화 매트릭스

### 4.1 정치 성향 × 기사 카테고리 매트릭스

| 기사 카테고리 | 진보 (1-3) | 중도 (4-6) | 보수 (7-10) |
|------------|----------|----------|-----------|
| 경제 | 분배 효과, 서민 영향, 불평등 지표 강조 | 거시지표 + 개인 영향 균형 | GDP 성장, 기업 경쟁력, 투자 환경 강조 |
| 정치 | 복지 정책, 노동권, 소수자 보호 측면 | 여야 균형, 정책 실효성 | 안보, 경제 성장, 규제 완화 측면 |
| 기업 | 노동자 처우, 환경 영향, 갑을 관계 | 경영 성과 + 사회적 영향 균형 | 매출·영업이익, 경영 효율, 주주 가치 |
| 부동산 | 임차인 보호, 주거 불평등, 공공주택 | 공급-수요 균형 분석 | 투자 수익, 규제 완화 필요성, 민간 공급 |
| 사회 | 사회 구조적 원인, 정부 책임 | 사건 팩트 + 다양한 시각 | 개인 책임, 법질서, 안전 강화 |

**적용 기준:** 정치 성향은 사실(fact)을 바꾸지 않고, **어떤 측면을 먼저 언급하고 얼마나 강조하는지**만 조정

| 구조 요소 | 격식체 (formal) | 비격식체 (casual) | 간결체 (concise) | 상세체 (detailed) |

이창용 한은 총재는 "물가 상승세가 예상보다 지속되고 있어

### 프로파일 A: 30대 직장인 투자자 (진보 성향)

**[한 줄 핵심 요약]**
한은이 또 금리 올렸어요 - 내 대출이자, 주식, 집값에 어떤 영향일지 같이 확인해봐요!

**[본문]**
한국은행이 기준금리를 0.25% 올려서 이제 연 3.75%가 됐어요. 6명 중 5명이 찬성했을 만큼 거의 확실한 결정이었죠.

문제는 이게 서민 대출자들한테 꽤 부담이 된다는 거예요. 이미 3억 주담대 가진 30대라면 이자가 월 6-7만원 정도 더 나올 수 있거든요. 특히 변동금리로 갈아탄 분들은 지금 시뮬레이션 한 번 해보시는 게 좋을 것 같아요.

미국 연준도 이달 말 추가 인상을 고민 중이라서, 환율 변동도 당분간 불안할 수 있어요. 수출 기업 주식 보유 중이라면 달러 강세 영향도 체크해보세요.

한편으로 고금리 혜택을 받는 서민들도 있어요 - 예금 금리가 올라가니까 정기예금 넣어두신 분들은 수익이 좀 더 늘어날 수 있답니다.

**[한 줄 핵심 요약]**
"금리"가 올랐다는 뉴스, 나한테 왜 중요한지 쉽게 설명해드릴게요!

**[본문]**
오늘 한국은행이 "기준금리"를 올렸다는 뉴스가 떴어요. 근데 이게 뭔 말인지 잘 모르겠죠?

**기준금리**란 쉽게 말해 '돈의 가격'이에요. 마트에서 물건 가격이 오르면 덜 사듯이, 돈 빌리는 비용(금리)이 오르면 사람들이 대출을 덜 받게 되는 거예요. 한은이 물가가 너무 빨리 오르는 걸 막으려고 이 금리를 올린 거예요.

취준생인 나한테 직접적인 영향은요?
- 기업들이 돈 빌리기 더 어려워져서 **투자·채용을 줄이는 경향**이 있어요
- 특히 스타트업이나 IT 기업들이 영향을 많이 받아요
- 당장 취업 준비할 때 안정적인 대기업이나 공기업 지원도 고려해볼 만해요

반대로 좋은 점도 있어요. 취업 후에 적금이나 예금 금리가 올라가니까, 첫 월급 모을 때 이자를 더 받을 수 있답니다!

미국도 같은 방향으로 금리를 올릴 예정이라, 글로벌 IT 기업들의 채용 상황도 같이 지켜보면 좋을 것 같아요.

**[구직자에게 중요한 포인트]**

- 인턴십으로 경력 쌓으면서 채용 시장 회복 대비 추천
- 금융·은행권은 고금리 수혜로 채용 오히려 늘어날 수 있음

## 부록: 프로파일 업데이트 전략

### A. 암묵적 피드백을 통한 프로파일 진화

| 행동 신호 | 프로파일 업데이트 |
|---------|--------------|
| 기사 끝까지 읽음 (80%+) | interest_categories 가중치 +1 |
| 빠르게 스킵 (5초 미만) | 해당 카테고리 가중치 -1 |
| 공유하기 | 강한 관심 신호, 가중치 +2 |
| "더 쉽게 설명해줘" 탭 | knowledge_level 하향 조정 고려 |
| "더 자세히 보기" 탭 | knowledge_level 상향 조정 고려 |

### B. 프로파일 신뢰도 관리

```
초기 온보딩: confidence = 0.5 (기본값)
기사 10건 읽음 후: confidence += 0.1
기사 50건 읽음 후: confidence = 0.9 (안정화)

신뢰도 < 0.6: 온보딩 질문 재확인 권장
신뢰도 >= 0.8: 자동 미세조정 활성화
```

### C. 프라이버시 보호 원칙

- 정치 성향 데이터는 개인 식별 불가능한 해시 처리
- 프로파일 데이터는 90일 주기 재동의 요청
- 사용자가 언제든 전체 리셋 가능 ("처음부터 시작하기" 옵션)
- 서드파티 공유 금지, 내부 개인화 목적으로만 사용
